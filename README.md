v4l2web
=======
Web server for V4L2 interface

The HTTP server use mongoose to give access to Vindeo4Linux interface and the web interface use AngularJS.

License
------------
Domain public 

Requirements
------------
 - libjsoncpp-dev 
 - libv4l-dev
 
Build
------- 
	make
